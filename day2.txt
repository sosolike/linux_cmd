file命令：
    功能：查看文件类型
    参数：
        file file_path;
        -b  :忽略文件名
        -i  :打印文件mini类型
        -z  :尝试解读压缩文件的内容
        -L  ：直接显示符号链接所指向的文件的类别
 find命令：
  功能：在指定目录查找指定文件。任何参数之前的内容都会被当做要查找的目录，如果不设置则默认在当前目录下查找文件；
  参数：
      -name  使用文件名查找文件
      -size  使用文件大小来查找文件
      -empty 查找空文件
      -ctime n 过去n天被修改过的文件
      -cnewer file 比file文件更新的文件
      -cmin n 在过去n分钟内被修改过的文件
      -atime n 在过去n天被读取过的文件
      -amin n 在过去n分钟内被读取过的文件
      -type type 列出指定类型的文件：
      -perm xxx 文件属主具有 指定权限的文件
            类型有：f 普通文件
                   c c文件
                   l 符号连结；
                   s socket
    ln命令：
      功能：为某一个文件在另外一个位置建立一个链接；
      软硬连接：
        软链接：
          1.软链接，以路径的形式存在。类似于Windows操作系统中的快捷方式
          2.软链接可以 跨文件系统 ，硬链接不可以
          3.软链接可以对一个不存在的文件名进行链接
          4.软链接可以对目录进行链接
        硬链接：
            1.硬链接，以文件副本的形式存在。但不占用实际空间。
            2.不允许给目录创建硬链接
            3.硬链接只有在同一个文件系统中才能创建
      参数：
          -b  删除，覆盖以前建立的连接；
          -d  允许超级用户制作目录的硬链接
          -f  强制执行
          -i  交互模式
          -n  把符号连结视为一般目录
          -s 软连接
          -v 显示详细处理过程
      方式：
          ln  -options 源文件  创建的连接文件
